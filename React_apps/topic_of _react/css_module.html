üß† What Happens Behind the Scenes

üìä CSS Modules vs Global CSS

Feature Global CSS CSS Modules
Scope Global (affects everything) Local (scoped to one component)
Class Naming Simple names (e.g. .title) Generated unique names (Card_title__abc12)
Conflict Risk High None
Ease of Maintenance Hard in large apps Easy per component
Import Style import './App.css' import styles from './Card.module.css'
